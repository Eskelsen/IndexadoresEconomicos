# Indexadores Econômicos

APIs públicas para consulta de indexadores econômicos brasileiros (IGP-M, IPCA, salário mínimo e outros), atualizados a partir de fontes oficiais.

## Endpoints
### Raiz
https://api.mfwks.com/indexadores/
### Docs
https://api.mfwks.com/indexadores/docs
### IPCA
https://api.mfwks.com/indexadores/ipca
### IGP-M
https://api.mfwks.com/indexadores/igpm
### Salário mínimo
https://api.mfwks.com/indexadores/salario-minimo
